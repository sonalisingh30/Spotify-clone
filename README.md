# Spotify Clone

This is a Spotify clone created using the following technologies:

- React
- Tailwind CSS
- Vite
- Redux
- React Icons

## Hosted Link

The Spotify Clone is hosted on Netlify. You can access it using the following link:

[Spotify Clone](https://sonali-spotify-api.netlify.app)
